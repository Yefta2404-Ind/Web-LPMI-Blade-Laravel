## ūüéĮ Tentang Proyek

**LPMI Campus CMS** adalah platform manajemen konten yang dibangun khusus untuk **Lembaga Penjaminan Mutu Internal (LPMI) Universitas Gunung Kidul**. Sistem ini dirancang untuk mempermudah pengelolaan konten website kampus ‚ÄĒ dari berita, galeri, hingga struktur organisasi ‚ÄĒ secara terpusat, aman, dan mudah digunakan oleh berbagai peran pengguna.

> ūüí° *Tidak perlu sentuh kode. Cukup login, kelola, dan publish.*

## ūüöÄ Instalasi

> Pastikan sudah menginstall **PHP 8+**, **Composer**, dan **MySQL** di mesin kamu.

```bash
# 1. Clone repository
git clone https://github.com/Yefta2404-Ind/lpmi-campus-cms.git
cd lpmi-campus-cms

# 2. Install dependencies
composer install

# 3. Konfigurasi environment
cp .env.example .env
php artisan key:generate

# 4. Setup database
# Edit .env ‚Üí sesuaikan DB_DATABASE, DB_USERNAME, DB_PASSWORD
php artisan migrate

# 5. Jalankan server
php artisan serve
```

ūüéČ Buka **http://localhost:8000** di browser kamu!
